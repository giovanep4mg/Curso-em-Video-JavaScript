No website, numa página de internet, existe 3 tecnologia essenciais.
HTML = que é o conteúdo, o que você quer mostra, texto, imagens.
CSS = que é o estilo, cores, tamanho, deixar mais bonito para visualizar..
JS = vai fazer a interação entre o usuário e a página.


HTML = Conteúdo.
CSS = Estilo.
JS = Interação.
